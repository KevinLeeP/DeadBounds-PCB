# DeadBounds

DeadBounds is a Wolfenstein and DOOM-inspired first-person shooter game where the objective is to eliminate zombies to survive as long as possible. However, as time progresses, more zombies spawn at random locations on the map at faster rates. The 3D rendering seen in our game is done using raycasting, which is an early rendering technique used to create 3D perspective on a 2D map, and allowed us to run everything off of a MSPM0G3507 Launchpad controller. You interact with the game using two joysticks: one to move your character’s position on the map and the other to look around the x-axis. To shoot enemies, we have a button in the back to act as a trigger. And we also have a button in the front to change between languages mid-game. We also have LED health and ammo bars on the sides of the screen.
